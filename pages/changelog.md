All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## Unreleased - yyyy.mm.dd

### Added

- New main server based on Nakama.
- Added accounts stored on main server.
- Added support for GNU/Linux and MacOS.

### Changed

- Updated to Unity 6.
- Updated to Wwise 2021.
- Make use of better text rendering on user interfaces.
- Rewrote networking from scratch.
- Switched to a higher-resolution skybox.
- Made sabilisation for player on all the time.
- Made game window resizable.
- Reworked tutorial.

### Fixed

- Fixed integration with Discord and Steam.
- Fixed multiplayer not working without Steam.
- Fixed countless small bugs in the original.
- Compress textures for drastically reduced file sizes.

### Removed

- Removed singleplayer.
- Removed some third-party dependencies.
- Removed volumetric lighting temporarily.
- Removed server browser.

Based on Hellion 0.5.2.
